# @pie-players/pie-esm-player

ESM-based web component for rendering PIE assessment items in modern runtimes.

## Install

```bash
npm install @pie-players/pie-esm-player
```

## Use

```html
<script type="module">
  import "@pie-players/pie-esm-player";
</script>

<pie-esm-player></pie-esm-player>
```
