# @pie-players/pie-fixed-player

Pre-bundled PIE player web component optimized for fixed element sets and predictable runtime performance.

## Install

```bash
npm install @pie-players/pie-fixed-player
```

## Use

```html
<script type="module">
  import "@pie-players/pie-fixed-player";
</script>

<pie-fixed-player></pie-fixed-player>
```
