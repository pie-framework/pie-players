# @pie-players/pie-iife-player

IIFE-based web component for rendering PIE assessment items with dynamic bundle loading.

## Install

```bash
npm install @pie-players/pie-iife-player
```

## Use

```html
<script type="module">
  import "@pie-players/pie-iife-player";
</script>

<pie-iife-player></pie-iife-player>
```
