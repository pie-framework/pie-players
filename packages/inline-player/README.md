# @pie-players/pie-inline-player

Standalone PIE player web component that renders an item from a single packaged request.

## Install

```bash
npm install @pie-players/pie-inline-player
```

## Use

```html
<script type="module">
  import "@pie-players/pie-inline-player";
</script>

<pie-inline-player></pie-inline-player>
```
